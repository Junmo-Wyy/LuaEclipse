Introdução

O LuaEclipse é uma coleção de plug-ins para a plataforma Eclipse. Esta ferramenta se propõem a facilitar o desenvolvimento de aplicações que utilizem a linguagem Lua. Em um primeiro momento de forma standalone, e em seguida de forma integrada com uma variedade de outras linguagens.

Os plug-ins são extensíveis e podem ser utilizados como base para outras ferramentas. A pesar desta possibilidade, esta seção é resume-se em um manual de utilização do software e não de extensã do mesmo.

LuaEclipse é um software livre e usa a mesma licença do Lua 5.0.

Instalação

O processo de instalação do plug-in é muito simples, mas necessita de alguns pré-requisitos. O primeiro deles é que o usuário já possui instalada a versão 3.* da plataforma Eclipse. O Eclipse pode ser encontrado no endereço www.eclipse.org. Consequentemente será necessário a instalação prévia do SDK de Java para executar o Eclipse. Comestes pré-requisitos satisfeitos, seguimos com a instalação.

Existem duas formas de instalar o LuaEclipse.

  • A partir do download manual:
    • Faça o download da ultima versão do LuaEclipse no site LuaForge. Para instalar no Eclipse, basta descompactar o arquivo zip na pasta raiz da instalação do eclipse, por exemplo: "/bin/eclipse". Feito isto, inicialize\reinicialize o Eclipse e a instalação está concluída.

Primeiro Projeto Lua

LuaSQL é apenas uma camada abstrata de comunicação entre Lua e um sistema de banco de dados. Portanto, erros podem ocorrem em ambos os níveis: no cliente do banco de dados ou no driver LuaSQL.

Erros como comandos SQL mal formados, nomes de tabela desconhecidos etc., chamados erros de banco de dados, serão reportados pelo retorno de nil pela função/método, seguido de uma mensagem de erro enviada pelo sistema de banco de dados. Erros como parâmetros inválidos, ausência de conexão, objetos inválidos etc., chamados erros de API, são usualmente erros de programação e geram erros Lua.

Esse comportamento vale para todas as funções/métodos descritos neste documento, a menos que seja especificado de outra forma.

Editando Arquivo Lua

Um driver LuaSQL permite o uso da API LuaSQL com um determinado banco de dados. Para usar um driver, este deve ser carregado na tabela luasql. O exemplo abaixo 

require "luasql.odbc"

carrega o driver ODBC na tabela luasql. Note que pode-se ter mais de um driver carregado usando-se algo como: 

require "luasql.odbc"
require "luasql.oci8"
