Introduction

LuaEclipse is a collection of plugins for the Eclipse plataform. This tools goal is to facilitate the development of programs using Lua, first as a standalone program, and then integrated with other languages.

The plugins are extensible and can be used as base for other tools, but this section is a manual on using the software, not extending it.

LuaEclipse is a free software and uses the same license as Lua 5.0.

Installation

The installation proccess is very simple, but needs some pre-requisites. The first one is that the user has already installed the 3.2.* version of the Eclipse plataform. Eclipse can be downloaded at www.eclipse.org. As a consequence, it is necessary that Java SDK is installed to run Eclipse.

There are two ways to install LuaEclipse.

  • By manual download:
    • Download the latest version of LuaEclipse at LuaForge. To install Eclipse, just decompress the zip file in the root folder of Eclipse, for example: "/usr/local/eclipse". After that start Eclipse and the installation is complete. In some systems it's necessary to start eclipse with the "-clean" command line argument.
      To check if the installation was correct, go to the menu option Help->About Eclipse SKD and select the Plug-ins Details option and check if LuaEclipse is installed correctly. Look the the picture bellow:

  • Eclipse Software Updates
    • On the top menu, go to option Help->Find and Install, choose option "Search for new features to install" and click on "Next".
    • Click on "Add Remote Site", in field URL type http://luaeclipse.luaforge.net/site.xml.
    • Choose only LuaEclipse and click on "Finish". Then follow the installing steps

First Lua Project

To create a new Lua Project, click on menu option File->New->New Lua Project.

To include a new Lua script, select the Lua project wanted, then click with the right button and select the option Lua Actions->New Lua File

Now just start coding. ;)

Preferences and interpreters

Now the only preference option on Lua is the configuration of Lua interpreters to execute scripts. To set a interpreter go to Windows->preferences... and then the Lua->Installed Interpreters category, as in the following picture.

In this windows it is possible to add, edit or remove an interpreter. The configuration of an interpreter is basically the name that identifies it and the path to its Lua executable. On the first column of the interpreter list, there is an option to select one of them as the default interpreter to execute scripts. This option is used on the script execution configuration. After the necessary configurations are made, click on "Ok" and it will be saved.

Running a Lua file

LuaEclipse uses the Eclipse Platform Lauchers to run Lua scripts. To run a lua file, select the menu option Run->Run...
Create a new "Lua Standalone Application" launch configuration .

The File Tab: select the Lua project and file to be executed.

The Arguments Tab:configure the working directory, interpreter arguments and program arguments.

At Environment Tab: select or add a Lua interpreter that will run th script. At Variables field, add th environment variables needed for your script.

Press Run and view the script output at the "Console View".
