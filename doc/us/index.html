Overview

LuaEclipse is a collection of plugins developed for the Eclipse plataform that together make an IDE for the development of applications in the Lua programming language. In this environment it's possible to edit Lua scripts with syntax highlight, code completion, compilation errors, code and comment folding, scripts execution with a pre-configured interpretor, besides the tools that the eclipse plataform provides.

The main goal of the project is that new tools will be developed using he extension architecture that the Eclipse plataform provides and that LuaEclipse has available for extension of it's funcionalities.

Eclipse Plug-in

A plugin is the smallest part of the Eclipse plataform that can be developed and distributed separetly. Usually a tool is developed as one plugin, while others more complex tools are built as several plugins. Except for the main execution code of the Eclipse plataform, all functionalities are implemented as plugins.
All plugins are developed in Java. A plugin consists of Java classes packed in a JAR library and some files like images, XML files, native libraries, etc. Not all plugins have Java code, some of them are plugins holding help files and user manuals.

Status

LuaEclipse 1.0.0 (for Lua 5.0) is available for download.

The plugin has versions for Windows, Linux and OSX, for Eclipse 3.2.x

Credits

LuaEclipse 1.0

The 1.0 version was initially developed by Guilherme Martins as a graduation project in Computer Engeneering at PUC-Rio, through the fragments of the 0.5 version, for the newer version of the Eclipse plataform (3.*). Using a new approach on the architecture, it's expected a greater extensibility as the Eclipse plataform. It was supported by André Carregal and now joins the Kepler Project.
The implementation is compatible with Lua 5.0 and was implemented by Guilherme Martins, Thiago Ponte with great contributions from André Carregal and Danilo Tuler.

LuaEclise 0.5

LuaEclipse was developed in it's first version by Danilo Tuler, through the the Company Ideais on the second half of the year 2003. The first implementation was compatible with Eclipse 2.1. It was presented on the Eclipse Day at PUC-Rio. The source code was not distributed.

The development of LuaEclipse is now sponsored by Fábrica Digital, FINEP e CNPq.
